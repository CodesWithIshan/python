This is a Python based small game where the computer selects a number between 1 and 100 and the user has to guess it.

## Features
- **Error Handling:** The program will not crash if the user inputs a string by mistake.
- **Dynamic Hints:** The user is informed whether the number he entered is smaller or larger than the original number.
- **Attempt Tracking:** The number of times the user tries and wins is counted.

## How to Run
1. You must have Python installed on your PC.
2. Download the code.
3. Type in the terminal: `python main.py`

## What I learned (Learning Outcomes)
- Use of Python `random` module.
- `while` loop and conditional logic.
- Exception handling through `try-except` block.
